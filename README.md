# FlightInfoApp_Final

This project was created for **Sprint Week (Fall 2025)**.  
It includes two main parts:
1. **API Server (Spring Boot + MySQL)**
2. **CLI Client (Java)** that connects to the API using HTTP requests.

---

##  Features
- REST API built using **Spring Boot**
- Data stored in **MySQL**
- CLI app can:
    - View all flights
    - Search flights by destination
    - Get airports in a city
    - Get aircraft flown by a passenger
    - Get airports used by a passenger
- Simple text-based interface using Scanner

---

##  Technologies
- Java 21
- Spring Boot 3.2
- MySQL 8
- Maven
